# user-server

## Setup

```
$ yarn install
```

## Running

```
$ yarn start
```

## Testing

```
$ yarn test
$ yarn coverage
```

## Features

```
| Method | Purpose|
| -------| -------|
| GET /user | Return all users |
| GET /user/:id | Return user based on ID |
| PUT /user/:id | Update user based on ID |
| DELETE /user/:id | Remove user based on ID |
| POST /user | Create user |
```
